
# 🎮 Python Mini-Games (Project 2)

A collection of interactive, terminal-based Python games focusing on multiplayer logic and time tracking.

## 📂 Included Games

### 1. 🎲 Multiplayer Dice Game (`project2.py`)
A turn-based game where 2 to 4 players compete by rolling a six-sided die. It uses Python's `random` module to simulate the dice rolls and includes input validation to handle player turns and scorekeeping.

### 2. ⏱️ Timed Math Quiz (`timed math.py`)
A fast-paced math challenge that generates random arithmetic problems. It tracks the user's input and calculates the total time taken to complete the quiz.
